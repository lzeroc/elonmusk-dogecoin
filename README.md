# elonmusk-dogecoin
当马斯克发推文内容提到狗狗币时，自动在交易所购买

When Musk tweeted that Dogecoin was mentioned, it was automatically purchased on the exchange

# run
- npm i
- npm run start

# config update (open config.js)
- binance apiKey
- binance secret
- buy amount
- search keyword
